Pseudocódigo
Definição e Importância: Pseudocódigo é uma descrição de alto nível de um algoritmo que utiliza uma linguagem simples e próxima da linguagem humana. Ele é importante pois permite planejar e expressar soluções algorítmicas antes da implementação em uma linguagem de programação específica.

Fluxograma
Introdução e Representação Gráfica: Um fluxograma é uma representação gráfica de um algoritmo, utilizando símbolos padronizados para representar os diferentes elementos lógicos e de controle de fluxo.
Elementos Básicos:
Início/Fim: Representados por um óvalo.
Processo: Representado por um retângulo.
Decisão: Representada por um losango.
Conector: Utilizado para conectar diferentes partes do fluxograma.
